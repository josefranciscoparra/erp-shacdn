## Estructura de cada manual

Todos los manuales siguen la misma estructura:

1. **Qué es y para qué sirve**: Descripción del módulo
2. **Quién puede usarlo**: Roles y permisos necesarios
3. **Flujos principales**: Paso a paso de cada funcionalidad
4. **Pantallas y campos**: Descripción de la interfaz
5. **Preguntas frecuentes**: Dudas comunes
6. **Checklist de soporte**: Para verificar configuración

---

## Soporte

Si tienes problemas o dudas:

1. Consulta el manual correspondiente
2. Revisa las preguntas frecuentes
3. Usa el checklist de soporte para verificar configuración
4. Contacta a tu administrador de TimeNow
